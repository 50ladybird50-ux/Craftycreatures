## 🚀 Getting Started

1. **Clone or download** this repository
2. **Open `index.html`** in your web browser
3. **Customize** the gallery, colors, and content to match your brand

## ✏️ How to Customize

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #FF1493;      /* Change main color */
    --secondary-color: #00CED1;    /* Change secondary color */
    --accent-color: #FFD700;       /* Change accent color */
}
```

### Add Your Kandi Photos
Replace the emoji placeholders in the gallery with actual images:
```html
<!-- Change from emoji placeholder to image -->
<img src="path/to/your/kandi-image.jpg" alt="Kandi bracelet">
```

### Update Contact Email
Add a backend service (like Formspree or Netlify Forms) to handle form submissions in `script.js`.
